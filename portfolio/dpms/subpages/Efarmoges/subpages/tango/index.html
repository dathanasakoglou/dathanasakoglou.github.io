Thessaloniki Project Tango

Η τεχνολογία τρισδιάστατης χαρτογράφησης είναι πιά ευρέως διαδεδομένη, ακόμα και σε προσωπικές συσκευές. Όπως λαμβάνει χώρα η ψηφιοποίηση εγγράφων, φωτογραφιών και σημαντικών εντύπων από την πόλη της Θεσσαλονίκης, έτσι, μέσω αυτής της ιστοσελίδας και του Project Tango, πραγματοποιήται η ψηφιοποίηση των αρχαιολογικών χώρων της Θεσσαλονίκης. Αυτό θα δώσει τη δυνατότητα σε ανθρώπους με δυσκολίες στη μεταφορά αλλά και σε ανθρώπους από το εξωτερικό να περιηγηθούν στους χώρους αυτούς από τον υπολογιστή τους, ή ακόμα και από συσκευές εικονικής πραγματικότητας, που θα το κάνει ακόμα πιο ενδιαφέρων.

Project Tango

Μεσω του Project Tango και των υποστηριζόμενων συσκευών είναι δυνατή η χαρτογράφηση εσωτερικών χώρων σε πολύ σύντομο χρόνο απλά με το χρήστη να κινείται και με τη συσκευή να δημιουργεί σε πραγματικό χρόνο τρισδιάστατα μοντέλα του χώρου.

Προτεραιότητα μας η Προσβασιμότητα

Μέσω της εικονικής πραγματικότητας, που είναι πια ευρέως διαδεδομένη και προσβάσιμη απο όλους, άτομα με δυσκολίες στην κίνηση αλλα και άτομα εκτός πόλης θα μπορούν να επισκέπτονται τους αρχαιολογικούς χώρους ψηφιακά.

Συμμετοχή

Επικοινωνήστε μαζί μας, στην περίπτωση που εχετε στην κατοχή σας μια συσκευή project tango, για να συμμετάσχετε στην χαρτογράφηση των αρχαιολογικών χώρων της πόλης μας.
